Food App

YumRun – Food Ordering Web Application

A dynamic food ordering platform built with React.js, Redux Toolkit, and Tailwind CSS. Users can browse restaurants, search and filter dishes, add items to cart, and manage their orders seamlessly. The app fetches real-time data through API integration with optimized performance using lazy loading.

  • React.js with functional components and hooks
  • Redux Toolkit for state management (cart, filters)
  • Tailwind CSS for responsive and modern UI
  • Search & filtering functionality for dishes
  • Cart features: add, remove, clear items
  • Dynamic routing using React Router
  • API integration for live restaurant data
  • Lazy loading & code splitting for performance
